## üîç Project Overview

This repository performs an end-to-end exploratory data analysis (EDA) of water quality measurements (heavy metals, ions, bacteria/virus indicators, etc.) and provides visualizations and insights about factors associated with `is_safe` (0 = Unsafe, 1 = Safe). The work focuses on data understanding, cleaning, visualization, outlier detection, and summary insights ‚Äî no machine learning models are built.
